# Menu Manager

**Change top menu items.**

## Overview

Top menu items that can be managed:
- Home
- Dashboard
- People
- Spaces
- Calendar
- Members Map
- Events Map

Item settings:
- Show/hide to: All, Logged-in users, Admins only or None
- Label
- Icon
- Sort order
